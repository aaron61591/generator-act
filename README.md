# generator-act 

> [Yeoman](http://yeoman.io) generator


## 安装

```
$ npm install -g yo
$ npm install -g generator-act
```

## 使用

### 生成项目

```
$ mkdir my-project
$ cd my-project
$ yo act
```

### 添加页面

```
$ yo act:page <pageName>
```

其中 `pageName` 为页面名

### 添加国际化配置

```
$ yo act:i18n <language>
```

其中 `language` 为语言名称，如 `en`，`ru` 等
