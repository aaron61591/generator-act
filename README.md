# generator-act 

> [Yeoman](http://yeoman.io) generator


## 安装

```
$ npm install -g yo
$ npm install -g generator-act
```

## 使用

### 生成项目

```
$ mkdir my-project
$ cd my-project
$ yo act
```

### 生成页面文件

```
$ yo act:view <viewName>
```

其中 `viewName` 为页面名

### 生成组件文件

```
$ yo act:cmp <componentName>
```

### 生成国际化配置文件

```
$ yo act:i18n <language>
```

其中 `language` 为语言名称，如 `en`，`ru` 等
